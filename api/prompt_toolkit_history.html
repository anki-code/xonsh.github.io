Prompt Toolkit History Object (xonsh.prompt_toolkit_history)¶

History object for use with prompt_toolkit.

class xonsh.prompt_toolkit_history.LimitedFileHistory[source]¶

History class that keeps entries in file with limit on number of those.

It handles only one-line entries.

Initialize history object.

append(entry)[source]¶

Append new entry to the history.

Entry sould be a one-liner.

read_history_file(filename)[source]¶

Read history from given file into memory.

It first discards all history entries that were read by this function before, and then tries to read entries from filename as history of commands that happend before current session. Entries that were appendend in current session are left unharmed.

Parameters:

filename : str

Path to history file.
save_history_to_file(filename, limit=-1)[source]¶

Save history to file.

It first reads existing history file again, so nothing is overrided. If combined number of entries from history file and current session exceeds limit old entries are dropped. Not thread safe.

Parameters:

filename : str

Path to file to save history to.

limit : int

Limit on number of entries in history file. Negative values imply unlimited history.
